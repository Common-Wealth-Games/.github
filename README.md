## 📦 Featured Project: LocalShelf
**The definitive open-source, offline PC game launcher.**

LocalShelf is a lightweight, all-in-one launcher built for players who value privacy and simplicity. It stays out of your way and lets your hardware focus on the game.

* **⚡ Lightweight:** Minimal CPU and RAM footprint.
* **🌐 100% Offline:** No forced logins or "home phoning."
* **📄 JSON-Powered:** Your library database is stored in human-readable JSON files. You own your data.
* **🛠️ Fully Open Source:** [View the Repository Here]
